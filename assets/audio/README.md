# Audio assets

Đặt file nhạc nền tại đây với tên: `bg_music.mp3`

Nguồn nhạc miễn phí gợi ý:
- https://pixabay.com/music/ (miễn phí bản quyền)
- https://freesound.org/
- https://incompetech.com/

Nếu không có file, app vẫn chạy bình thường — chỉ là sẽ không có nhạc nền.
Các hiệu ứng click vẫn hoạt động vì dùng `SystemSound` của Flutter.
